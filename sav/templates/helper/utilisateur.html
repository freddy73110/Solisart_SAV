4. Page utilisateur

La page utilisateur regroupe toutes les informations relatives à un utilisateur enregistré dans my.solisart.fr.

Le nom de l'utilisateur est suivi par des icônes représentant son status:

  • Administrateur
  • Technicien
  • Installateur
  • Propriétaire

La carte pointe l'adresse de l'installateur en rouge si c'est un installateur et toutes les installations liées à l'utilisateur. En cliquant, sur les pointeurs jaunes, un popup apparaît avec un lien vers l'installation en question. Il est possible de repositionner le pointeur installateur avec en cliquer-déplacer.

Pour accéder directement à l'utilisateur sur my.solisart.fr, il suffit de cliquer sur le logo Solisart.

Le tableau regroupe le listing de toutes les installations liées à l'utilisateur avec des liens directs sur les installations et le propriétaire.

Dans l'ordre des onglets qu'il est possible de visualiser:

  • Description
  • Information Héraklès pour les installateurs
  • Tickets en relation avec l'utilisateur
  • Evaluation
  • Bilan SAV pour les commerciaux

4.1 Description

La partie général regroupe des informations sortie de la page utilisateur de my.solisart.fr. Elles sont quand même modifiables.

La partie détail regroupe des informations sorties de la page utilisateur de my.solisart.fr. Elles sont elles aussi modifiables. Le champs de choix sélectionnable "Client Héraklès" apparait que si l'utilisateur a au moins un status installateur. Il permet de faire le lien entre le SolisTools et Héraklès. Il est très utile pour savoir si l'installateur appartient à une COOP.

Si l'utilisateur appartient à Solisart (is_staff coché) et que c'est un commercial, il est possible de lui affecter une liste de département.

Le format de la liste est très important! Dés que le commercial a au moins un département d'affecté, son nom et son numéro de téléphone apparait sur la carte des installations/installateurs. La couleur est la couleur avec laquelle la zone du commercial sera cadrié sur la carte des installations/installateur. Il faut que la case "Recevoir rapport des tickets hebdomadaires" soit cochée pour que le commercial reçoive un bilan SAV par email.

4.2 Information Héraklès

Cette partie concerne uniquement les utilisateurs avec le status installateur. Si l'installateur a le champ "Client Héraklès" complété Toutes les informations concernant ce client seront remontés directement depuis Héraklès.

Si vous appuyé sur le bouton "localiser avec l'adresse", le SolisTools récupère les coordonnées de géolocalisations avec l'adresse depuis internet et matérialise la géolocalisation avec un pointeur rouge.

4.3 Tickets

Cette partie répertorie tous les tickets dont utilisateur est lié.

4.4 Evaluation

Tous les utilisateurs peuvent être évalués de 0 étoile (mauvais) à 5 étoiles (Excellent) en fonction de différents critères. Cette évaluation peut être suivi temporellement.

Pour connaître le contenu de l'évaluation sur un critère, il suffit de passer la souris au dessus de la barre représentant l'évaluation.

Vous pouvez évaluer un utilisateur en cliquant sur le bouton "Ajouter une évaluation".

4.5 Bilan SAV

Cette partie est affiché uniquement pour les utilisateurs commerciaux (ayant une liste de département lui étant affectée). Elle liste le bilan SAV sur toute la zone du commercial pour les x derniers jours.
