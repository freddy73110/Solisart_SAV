1. Tronc commun

1.1 Navigation

Cette barre de navigation permet d'aller facilement sur toutes les différentes applicationS de SolisTools. Elle comporte:

  • Pour retrouver rapidement une installation ou une personne
  • Pour le transfert de données en base de données et lecture de json installation
  • Pour la gestion de cadencement de production
  • Pour la gestion de lot
  • Pour le suivi les tickets ouverts
  • Pour localiser les installations et leurs installateurs
  • Pour le traitement statistique des tickets
  • Pour avoir un gestionnaire de version de document

1.1.1 Recherche d'installation

Le solistools est capable de retrouver une installation et/ou une personne à partir d'une partie de :

  • Nom
  • Prénom
  • Numéro de téléphone
  • Email
  • N° d'installation
  • Code postal
  • Ville
  • Données Héraklès:
    • Bon de livraison
    • Bon de Commande
    • Devis

1.1.2 Transfert et conversion de données

Cette page comporte 3 grosses parties:

  • Convertion des json de l'installation vers schéma exe, hydro et fichier csv de configuration d'installation

    • {{admin}}
      • {%endfor%}
  • Exporter les prix d'Héraklès vers le site commercial pour l'application de chiffrage en ligne

1.1.3 Gestion cadencement de production

Cette page permet d'avoir et de faire un suivi de production sous format de diagramme de Gantt ou de liste d'événement à faire. Elle est relié directement à la création de carte de carte de régulation et la traçabilité.

1.1.4 Traçabilité et gestion de lot

Cette page renseigne tous les lots qui ont été créé et elle permet de suivre leur validation et leur impact sur les installations en cas de défaillance d'un organe.

1.1.5 Suivi des tickets ouverts

Cette page renseigne tous les tickets qui ne sont pas encore fermés.

1.1.6 Localisation installations - installateur

Cette page permet de visualiser:

  • Tous les installations dont les coordonnées de geolocalisées sont rentrées
  • Tous les installateurs dont les coordonnées de geolocalisées sont rentrées
  • De visualiser les différentes zones de chaque chargée d'affaire
  • D'avoir toutes les informations (chargé d'affaire, altitude, température de base, ...) d'une adresse

1.1.7 Statistiques sur les tickets (Nombres, catégorie, coût associé), suivi des non-conformités

Cette page renseigne tous les tickets qui ne sont pas encore fermés.

1.1.8 Suivi des versions de document

Cette page permettait de connaître les différents version de documents dans le temps.

  • Changer son thème sur le SolisTools
  • Modifier son mot de passe
  • Ajouter un nouvel utilisateur avec au moins les mêmes droits
