5. Transfert entre base de donnée

Cette page permet de récupérer ou d'envoyer des données pour rafraichir les bases de données entre elles.

Elle comporte 3 onglets:

  • Générer schéma à partir json
  • Importer depuis my.solisart.fr
  • Exporter vers site commercial

5.1 Générer schéma à partir json

A partir du json généré sur la schémathèque en ligne, le Solistools permet de visualiser et de télécharger simultanément:

  • En cliquant sur hydraulique: le schéma hydraulique
  • En cliquant sur Exe: le schéma de connexion du module
  • En cliquant sur fiche de prod : la fiche de production
  • En cliquant config: le fichier csv injectable directement à une installation.

Dans le dernier cas, pour personnaliser l'installation, il faut ouvrir le fichier csv, et remplacer dans la ligne 3, ADefinir par le numéro de l'installation (SC2M202....) et ligne 4 install par le nom du pripriétaire en majuscule (DUPONT).
