2. Page de recherche

Il existe 3 groupes de recherche:

  • par information sur l'utilisateur
  • par information sur l'installation
  • par information Héraclès

Les informations rentrées pour faire une recherche peuvent-être une partie du mot. Par exemple, si vous rentrez "mar" dans le nom d'utilisateur, il sortira tous les noms avec "mar": "Martin", "Martinez" mais aussi "Pamars".

Par groupe de recherche, vous pouvez ajouter plusieurs niveau comme "Module" = "2024" et code postal = "74", pour obtenir pour les installations de 2024 situé dans le département 74.

Pour lancer une recherche, après avoir effectué votre saisi, il vous suffit d'appuyer sur votre touche "Entrée".

2.1 Recherche par utilisateur

Cette recherche sortira tous les utilisateurs et les installations en rapport avec la partie de mot clé entrée.

Pour le cas du nom, le Solistools recherche également dans id de l'installation (surtout valable pour les anciennes installation avec ID = NOMdeFAMILLE).

2.2 Recherche par installation

Cette recherche sortira tous les installations en rapport avec la partie de mot clé entrée.

Pour le code postal et la ville, le Solistools recherche dans tous les renseignements enregistrés dans my.solisart.fr dans la rubrique "information".

Pour la recheche par module, le Solistools recherche dans toutes les noms d'installations. Si vous tapez, "202449" vous obtiendrez toutes les installations de 2024 faites la 49ième semaine.

2.3 Recherche par données Héraklès

Cette recherche sortira tous les installations ou les commandes en rapport avec la partie de mot clé entrée.

Si vous rentrez une partie de numéro de devis ou de bon de commande, le SolisTools vous sortira toutes les installations avec un ticket qui a un devis ou bon de commande relié qui comporte le mot clé saisi.

Pour la recheche par commande, le Solistools sortira soit un lien vers le bon de commande dans la partie Production soit la commande n'est pas finalisée soit l'installation liée au bon de commande.
